Version 1.3 (2010-06-12)
============================================
Project:
 * Switch project build system to CMake
 * Auto-generation of project files/variables,
   eg. shared libraries, manifest files etc.
 * Automatic generation of browser-specific
   packages (Firefox Add-on, Chrome Extension
   and Opera Widget)
Plugin:
 * Auto-generation of project files using CMake
   (Linux and Windows compatible)
 * Significant clean-up in directory structure
   and platform-dependent files
Extension:
 * Redesign - divide layout into local services
   and external info (be ready for more features,
   eg. external services and local info)
 * New set of icons (by Rafal Ramatowski)
 * Add settings tab (based on LocalStorage)
 * Possibility to autorun services on startup
 * Selectable extension tab as front-page
 * Selectable jQuery themes in settings
   (Cupertino, South Street and Sunny)
 * Rewrite plot.js and other files into OOP
 * Fix long output in service consoles
 * Rewrite lost packet analysis into regexp
 * Improve visibility of console time values
 * Fix IPv6 address inside the http protocol
 * Automatic toolbar icon in Firefox
 * "Google.com - JSAPI" parser
   (countryCode, country, city, region,
    latitude, longitude)
 * Asynchronous Google Map with location
   markers

Version 1.2 (2010-03-31)
============================================

Extension:

 * Services parser (data from remote servers)

    region, latitude, longitude)

   (externIpv4)

   (externIpv4, hostname)

   (externIpv4, hostname)
 * "RADb (Merit Network Inc.)" parser
   (route, provider)
 * Google map (init latitude and longitude)
